# My Personal Website

## Description
This is a personal website created as part of a web development course. It showcases my projects, provides information about me, and includes a contact form for inquiries.

## Technologies Used
- HTML5
- CSS3
- GitHub Pages for hosting

## Features
- Responsive design
- Navigation menu
- Projects section with details and a table
- Contact form

## Installation
To view the website locally, follow these steps:
1. Install XAMPP.
2. Place the project folder in the `htdocs` directory.
3. Start the Apache server from the XAMPP control panel.
4. Open a web browser and navigate to `http://localhost/yourProjectFolder`.
